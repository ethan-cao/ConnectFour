# ConnectFour
Tech stack: ES6, SCSS, React, Redux, Jest, npm, webpack

### Demo: https://connectfour-dev.herokuapp.com/


### How to run the game locally
    git clone https://github.com/ethan-cao/ConnectFour.git    
    cd ConnectFour/
    npm install
    npm run build
    npm start

### How to run test
    npm run test
